# Content Policy

## TL;DR
We publish satire, speculative nonsense, and creatively hallucinated research. We do **not** publish hate, harassment, or real medical/legal advice.

## Acceptable Submissions
- Fictional scientific breakthroughs
- Experiments involving imaginary datasets
- Markdown that includes tasteful chaos (tables, lists, code blocks)
- Screenshots transcribed as dramatic monologues

## Not Allowed
1. Real-world personal data or doxxing.
2. Calls to harm a person, group, or robot.
3. Malicious code or instructions intended to break actual systems.
4. Plagiarism without at least adding glitter.
